# Career AI Project 🚀

AI-based Resume Screening and Job Matching System using Machine Learning and NLP.

## Features
- Resume parsing and skill extraction  
- Job-role matching using TF-IDF + Cosine Similarity  
- NLP using spaCy / NLTK  
- Streamlit web interface  
- Optional SQLite database  

## Tech Stack
Python, NLP, scikit-learn, TF-IDF, Streamlit, SQLite

## Run Project
```bash
pip install -r requirements.txt
streamlit run app.py
